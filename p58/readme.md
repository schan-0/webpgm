# `p58` WEEK2 REPORT 01
HTML 문서를 구성하는 기본적인 태그들을 알아보았습니다.
|Tag|Description|
|:---:|:---|
|`<head>`|메타데이터 및 CSS, JS 스크립트를 작성하는 공간|
|`<style>`|CSS코드 작성 공간|
|`<script>`|JS코드 작성 공간|
|`<body>`|본문|
|`<title>`|탭에 표시되는 내용|
|`<h1>`|문단 제목. `<h6>`까지 있음|
|`<p>`|문단|
|`<div>`|블록 태그|
|`<span>`|인라인 태그|
|`<img>`|이미지 첨부|
